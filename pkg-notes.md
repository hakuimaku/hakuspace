# Hakuspace Package Categories

Note: This file is a reference for the package categories used in `pkg.txt`. It provides an overview of the different types of packages included in the Hakuspace setup, organized by their functionality and purpose. This helps users understand what each package does and how it contributes to the overall desktop environment.
